01 · Visão Geral

O que é o ChargeGrid Intelligence

ChargeGrid Intelligence é um sistema de gerenciamento de infraestrutura de recarga de veículos elétricos voltado ao setor comercial. A plataforma centraliza o controle de demanda energética, o gerenciamento de eletropostos e a aplicação de políticas de tarifação dinâmica em uma única base de dados.

O sistema atende simultaneamente a três perfis distintos de usuário — clientes finais (residencial e comercial), gestores de estabelecimentos parceiros e administradores globais da GoodWe — garantindo consistência operacional e rastreabilidade completa das sessões de recarga.

A implementação utiliza Python puro com arquitetura MVC, sem dependências externas, o que facilita a execução em qualquer ambiente com Python 3.7 ou superior instalado.

02 · Problema e Solução

Por que este sistema existe

A infraestrutura comercial de recarga de VEs enfrenta três lacunas críticas que comprometem a escalabilidade do setor.

Lacuna

Controle de potência fragmentado

Sem mecanismo centralizado para gerenciar distribuição de energia entre múltiplos carregadores simultaneamente.

Solução

Seleção e cálculo por nível

Carregadores 7kW / 11kW / 22kW com cálculo automático de tempo e energia por sessão.

Lacuna

Ausência de registro estruturado

Sessões de recarga sem rastreabilidade impossibilitam faturamento preciso e auditoria.

Solução

Histórico centralizado

Cada sessão é registrada em session_history com potência, energia, custo e duração.

Lacuna

Tarifação estática e inadequada

Preços fixos ignoram variação de demanda e não incentivam distribuição de carga ao longo do dia.

Solução

Tarifação dinâmica por horário

R$ 1,60/kWh (normal) e R$ 2,10/kWh nos horários de pico (18h–21h).

03 · Módulos

Funcionalidades por perfil

O sistema é organizado em quatro módulos independentes, cada um projetado para um tipo específico de usuário.

Módulo Comercial B2C · Recarga Pública

Simula a experiência completa de recarga em postos parceiros da GoodWe, desde a seleção do carregador até o relatório final da sessão.

Seleção de potência 7/11/22kW Tarifação dinâmica por horário Barra de progresso visual Registro automático de sessão Relatório final formatado
Módulo Residencial Smart Home

Experiência simplificada para usuários com carregador doméstico, com foco em economia e telemetria sem complexidade de pagamento.

Onboarding com validação de modelo Dashboard de economia Comparação elétrico vs. gasolina Informações técnicas do carregador 7 modelos de VE suportados
Módulo Estabelecimento B2B · Gestores

Painel de controle para proprietários de estabelecimentos parceiros monitorarem hardware, faturamento e desempenho dos carregadores instalados.

Relatórios: diário / semanal / mensal Status de hardware em tempo real Gestão de manutenção Ranking de rendimento financeiro
Módulo Admin Master GoodWe · Global

Visão consolidada da infraestrutura nacional para administradores da GoodWe, com controle sobre tarifas, royalties e saúde da rede.

KPIs de disponibilidade nacional Ajuste global de tarifas Cálculo automático de royalties Filtro de dispositivos críticos

05 · Instalação

Pré-requisitos e configuração

O projeto não possui dependências externas. Tudo que é necessário é o Python instalado.

1

Verificar instalação do Python

Requer Python 3.7 ou superior.

2

Obter o projeto

Via Git ou download direto do arquivo ZIP.

3

Executar diretamente

Nenhuma instalação adicional é necessária.

bash · verificar python
python --version
# ou
python3 --version
bash · clonar repositório
git clone https://github.com/seu-repositorio/chargeGrid-intelligence.git
cd chargeGrid-intelligence
